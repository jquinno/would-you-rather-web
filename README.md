# Would You Rather - Voting Experiment App

A Next.js application that allows users to participate in "Would You Rather" style voting experiments. Users can vote between pairs of items, and administrators can create and manage experiments through an admin interface.

## Features

- **Interactive Voting**: Users can vote between pairs of items in a card-based interface
- **Experiment Management**: Admin interface to create and manage voting experiments
- **Real-time Data**: Firebase Firestore integration for real-time vote tracking
- **Responsive Design**: Built with Tailwind CSS and shadcn/ui components
- **Analytics**: Vote statistics and experiment analytics
